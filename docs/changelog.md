## [Unreleased]

### Added
- Initial release of pytest-routes
- Route discovery for Litestar, FastAPI, and Starlette
- Property-based testing with Hypothesis
- Response validation with StatusCodeValidator, ContentTypeValidator, JsonSchemaValidator
- OpenAPI schema-based route extraction
- Custom strategy registration API
- pyproject.toml configuration support
- Comprehensive CLI options

### Documentation
- Sphinx documentation site with shibuya theme
- API reference with autodoc
- Framework-specific guides
- Contributing guide
